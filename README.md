# nsfw-bot
A Discord bot that posts NSFW images.

## How to run
- In the main directory add a file named `config.properties` and fill it with a bot token and a default prefix
```proprties
[DEFAULT]
token=birdup
prefix=!
```
- Run `launcher.py`
